# Learnings

Corrections, insights, and knowledge gaps captured during conversations. Source for dreaming (`memory-core` reads this on each cycle and promotes patterns to `MEMORY.md`).

**Categories**: correction | insight | knowledge_gap | best_practice

**When to log here**:
- Owner corrects the agent: "no, actually..." → category `correction`
- Agent realizes its model knowledge is stale → category `knowledge_gap`
- A better approach is discovered for a recurring task → category `best_practice`
- A pattern emerges that's worth remembering → category `insight`

**Entry format** (append at the top of the list below):

```markdown
## [LRN-YYYYMMDD-XXX] category

**Logged**: ISO-8601 timestamp
**Priority**: low | medium | high | critical
**Status**: pending
**Area**: frontend | backend | infra | tests | docs | config

### Summary
One-line description of what was learned.

### Details
Full context: what happened, what was wrong, what's correct.

### Suggested Action
Specific fix or improvement to make.

### Metadata
- Source: conversation | error | user_feedback
- Related Files: path/to/file.ext
- Tags: tag1, tag2

---
```
