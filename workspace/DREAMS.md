# DREAMS

Human-readable narrative log of dream cycles. Written by the REM phase of `memory-core` dreaming. Each entry is a short diary-style reflection — what the agent noticed in recent signals, patterns it saw, questions it's tracking.

**This file is reflective writing, not a promotion source.** Editing it doesn't change long-term memory. For that, see `MEMORY.md` and the `.learnings/` log files.

To read the last week of dreams: `tail -200 DREAMS.md` or grep by date.

To skip a dream cycle: `/dreaming off` (slash command) or set `dreaming.enabled = false` in the `memory-core` plugin config.
